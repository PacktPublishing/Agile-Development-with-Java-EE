VIDEO PLATFORM



Help
This application is a video platform which allows video uploading, streaming and categorization.
Click on Add Category icon to create a new category for your videos. Provide a title and description for the category in the text fields.
To add a video to a given category, Click on 'Choose file' button to select a video file, and 'Upload' button to upload the video file. To modify a category title or description, Edit the text and press Enter.
You can add multiple video files to a category.
Click on icon to delete the category.
